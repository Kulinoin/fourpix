# Capture Guideline 4Pix Studio

Status: active guideline for MVP capture quality.

## Tujuan

Membuat input foto lebih konsisten agar local background removal lebih stabil.

## Standar Awal

- Gunakan background putih atau terang bersih.
- Gunakan pencahayaan rata dari depan atau kiri-kanan.
- Hindari backlight.
- Hindari bayangan keras di belakang kepala, rambut, leher, dan bahu.
- Jaga jarak subjek dari background agar tepi rambut/bahu tidak menyatu dengan bayangan.
- Gunakan overlay guide/template untuk menjaga posisi kepala, dagu, leher, dan bahu.
- Jika memungkinkan, gunakan warna pakaian yang tidak terlalu mirip dengan background capture.

## Catatan

Guideline ini bukan pengganti AI/model. Ini adalah standar input agar model lokal 4Pix bekerja lebih optimal.
