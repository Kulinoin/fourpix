# 4Pix Studio UI Visual Lock

Status: Locked for MVP visual direction
Scope: 4Pix Studio web MVP

## 1. Visual Direction

4Pix Studio uses a clean photo-studio and camera-tech visual direction.

Approved visual personality:

- clean photo studio
- camera/liveview first
- modern tech
- slate / deep navy / cyan / electric blue
- crisp contrast
- focused workspace
- subtle glass/gradient effect
- not warm amber/orange
- not Gudange-like admin dashboard visual

## 2. Brand Feel

4Pix Studio should feel like a lightweight local camera studio app, not a warehouse/admin dashboard.

Keywords:

- studio
- camera
- liveview
- precision
- template-aware
- local-first
- future-ready

## 3. Color Direction

Primary palette:

- Deep navy: #07111f
- Slate: #101827 / #182235
- Electric blue: #2563eb
- Cyan: #06b6d4
- Sky: #38bdf8
- Soft panel: rgba(15, 23, 42, 0.72)
- Light surface: #f7fbff

Avoid for 4Pix MVP UI:

- warm amber
- orange-heavy accent
- cream warehouse/admin look
- Gudange-like visual treatment

## 4. Layout Direction

- Live camera should become the main visual center.
- Cards can be rounded but should feel precise and modern.
- Background may use cool gradients, subtle grid, camera-frame lines, or glass panels.
- UI should support future overlay/liveview workflow.

## 5. Slice Rules

Every future slice must preserve this visual direction unless the user explicitly changes it.

Do not overwrite global CSS with Gudange-like warm amber theme.
